The `compute_correlations_cli.py` script is used to compute CCC, Pearson, and Spearman correlations across all GTEx v8 gene pairs.

It generates three log files for each method:
- `gtex-var_pc_log2-{method}.terminal.log`: Terminal output with debug information.
- `gtex-var_pc_log2-{method}.progress.log`: Progress output.
- `gtex-var_pc_log2-{method}.root.log`: Root output without debug information.

Please refer to the script to set the input parameters.
